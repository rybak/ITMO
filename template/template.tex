\section{Проблемы и ограничения}
\begin{frame}
	\frametitle{Проблемы}
	\pause
	\begin{itemize}
		\item beamer не поддерживает использование latex + dvipdfmx, а pdfLatex не умеет рисовать label в картинках сделанных в METAPOST
		\pause
		\item Как решить эту проблему?
	\end{itemize}
\end{frame}

\begin{frame}
	\frametitle{Простое решение}
	Использовать TikZ для рисунков и pdflatex для сборки презентации.
\end{frame}

\begin{frame}
	\frametitle{Сложное решение}
	\begin{itemize}
		\item Использовать latex + dvips + ps2pdf, если нужны подписи в METAPOST
		\pause \item Но появляется вторая проблема: latex + dvips + ps2pdf не поддерживает вставку jpeg
	\end{itemize}
\end{frame}
